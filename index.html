MyUploads

Project Description:

This application provides a secure platform for file upload & storage, emphasizing robust security measures. Developed using Python with Flask and managed via MySQL Workbench, it integrates advanced authentication methods and user-friendly interfaces to ensure data protection.

Features Developed By Me:

  • Registration: Secure registration with validation and protected credential storage.
  • Captcha (hCaptcha): Blocks automated bot registrations.
  • Email Verification: Confirms user emails through verification links.
  • Login Alerts (ipify/ipinfo): Notifies users of logins from new IP addresses or devices.
  • Phone Number Verification (Twilio): Verifies phone numbers via SMS.
  • Phone Number OTP Login (Twilio): Secure login with OTP sent to verified phone numbers.
  • Email 2FA: Adds email-based two-factor authentication for login security.
  • SMS 2FA (Twilio): Uses SMS codes for additional login security.
  • App 2FA (PyOTP): App-based two-factor authentication with authenticator apps.
  • OAUTH (Google): Allows secure login and sign-up through Google.
  • Forgot Password (Email/SMS OTP): Secure password recovery via email or SMS.
  • Password Change: Secure password updates with verification.

Technological Components Used:

  • Flask Framework: Lightweight backend for efficient development.
  • Database Management: Managed with MySQL Workbench.
  • Front-end: HTML, CSS, and minimal JavaScript for a clean, responsive interface.

Features Developed by Other Members:

  • RBAC: Manages user permissions and group access.
  • Hashing & Encryption: Secures passwords and sensitive data.
  • Preventing SQLi and XSS: Uses parameterized queries and input validation.
  • Face Recognition: For secure and convenient logins.
  • HTTPS: Secure communication with SSL.
  • Password History: Prevents password reuse.
  • Session Management: Includes session locks and timeouts.
  • File Upload Security: Virus scanning and validation checks.
  • Input Validation: Ensures data integrity and security.

Sustainable Living Tips and Energy Tracker

Project Description:

The Sustainable Living Tips and Energy Tracker is an application designed to promote eco-conscious habits and efficient energy use. Leveraging Flask, HTML, CSS, and minimal JavaScript, the project integrates educational content with user-friendly interfaces. Implementations include Plotly for visualizing graphs in the Energy Consumption Tracker feature and WTForms for form creation and validation, including email validators for data accuracy in the Enquiry Form.

Features Developed By Me:

  • Energy Consumption Tracker: Empower users to log monthly energy usage, visualized through graphs for easy comparison using Plotly.
  • Enquiry Form: Facilitate engagement through an online form for inquiries about energy-saving practices, utilizing WTForms for form validation.

Technological Components Used:

  • Flask Framework: Ensures a streamlined user experience through Flask's simplicity.
  • Python Classes and Objects: Implements a structured data model with basic classes.
  • Storage with Shelve: Manages energy usage data efficiently using Shelve, with potential integration into existing infrastructure.
  • Basic Front-end Technologies: HTML, CSS, and minimal JavaScript for a clean, user-friendly interface.

Features Developed by Other Members:

  • Sustainable Living Tips: Engage users with quizzes for personalized guidance on sustainable practices.
  • Basic User Profiles: Allows users to create and manage profiles.
  • Online Transactions: Supports transactions for purchasing energy-efficient products.
  • Feedback System: Gathers valuable user feedback for continuous improvement.
  • Rewards System: Motivates users with a points-based system to redeem accumulated points for rewards.
  • Content Management: Ensures efficient content management for admins, keeping the platform relevant.
